# Line Coding Simulator

This project is a GUI-based simulator for line coding techniques used in computer networks.

## Features
- NRZ-L encoding
- Manchester encoding
- Transmission delay calculation
- Propagation delay calculation
- Signal visualization using Canvas

## Technologies
- Python
- Tkinter

## How to Run
```bash
python main.py
```

## Description
This project demonstrates how binary data is encoded into signals and how delay occurs in the physical layer.
